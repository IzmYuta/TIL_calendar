# Reactがもっと楽しくなるステップアップ講座

## 概要

- 再レンダリングの仕組みを知り、レンダリングを最適化
  - memo化
  - useCallback
  - useMemo
- 様々なCSSの当て方を知る
  - InlineStyle
  - CSS Modules
  - Styled JSX
  - styled components
  - Emotion
- ルーティング(画面遷移)の方法を学ぶ
  - ReactRouter
- コンポーネント分割
  - Atomic Design
- グローバルなstate管理を知る
  - useContext
  - Recoil
- TypeScript x React
  - 型の概念を取り込む
- カスタムフックを使ってレベルアップ
  - Viewとロジックを分離
- 実践的なアプリ開発
  - CSSフレームワーク(Chakra UI)の利用
